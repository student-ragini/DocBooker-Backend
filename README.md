DocBooker Backend:-

Backend API for the DocBooker Hospital Management System.

Features:

- JWT Authentication
- Patient Management
- Doctor Management
- Appointment Management
- Message Management
- MongoDB Integration
- Cloudinary Image Upload
- Secure Cookies

Tech Stack:

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Cloudinary

API Base URL:

https://docbooker-backend-2hzo.onrender.com

Environment Variables:

PORT=4000
MONGO_URI=mongodb_url
JWT_SECRET_KEY=secret_key
JWT_EXPIRE=7d
COOKIE_EXPIRE=7
CLOUDINARY_CLOUD_NAME=cloudinary_name
CLOUDINARY_API_KEY=cloudinary_api
CLOUDINARY_API_SECRET=cloudinary_secret

Installation:

npm install
npm start
